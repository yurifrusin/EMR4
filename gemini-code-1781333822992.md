### 4.1 Internal Document Linkage & Clinical Context
* **The Relationship:** Historical patient files contain explicit internal links connecting free-text clinical notes directly to the structured "Results" section within the same document. 
* **The Value:** This linkage provides the critical "clinical rationale" for the AI to digest. It maps the diagnostic pathway, showing how the doctor transitioned from a patient's presentation to an objective test result, and finally to a management decision or prescription.
* **Ingestion Strategy:** The Python extraction script must treat these internal links as relational data. Instead of parsing notes and results as isolated text blocks, it will bundle linked text-based pathology or radiology reports directly into the same JSONL encounter object, providing a complete contextual loop for RAG and fine-tuning.
